# 게시판



💡 [board](http://3.38.246.79:8080/)


## 실행 환경

- JAVA 11
- Spring Boot 2.6.3
- Gradle 7.3.3
- JPA
- H2 1.4.200
- BootStrap + Thymeleaf
- MySql 8.0.23
- AWS EC2/RDS


## 기능

- [ ]  **HOME** 
  + id, 제목, 작성자, 작성일(’yyyy-MM-dd HH:mm’)  
  + 홈버튼(board) 누르면 홈으로 이동


- [ ]  **글쓰기**
  + 제목, 작성자, 내용


- [ ]  **하나의 게시글 확인**
  + 제목, 글쓴이, 작성일(’yyyy-MM-dd HH:mm’), 내용


- [ ]  **글수정**
  + 내용 수정 가능
  + 수정 후 기본 홈 페이지


- [ ]  **글삭제**
  + 삭제 후 기본 홈 페이지
